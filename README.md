# Планировщик PlanUp
Автор проекта: Лунева Алина (я сама себе бэкендер и фронтендер)
Я хотела создать удобное и доступное приложение для планирования своих дел и для постановки целей, чтобы пользователь мог отслеживать свой прогресс выполнения и не забывать о том, что запланировал. Кроме того, чтобы пользователь не отходил от поставленной цели ему каждый день будет приходить сообщение о его важности, что является дополнительным источником мотивации. На рынке я такого ещё не встречала, поэтому я решиила, что это классная идея :)
Мой код представляет собой многофункциональный планировщик задач и целей на PyQt6 с системой уведомлений. 
Основные классы включают CalendarApp (главное окно с календарем и задачами), CustomCalendarWidget (кастомный календарь с визуальными маркерами дат), GoalTaskDialog (создание целей с валидацией) и систему диалогов для управления заметками и уведомлениями.
Интересные приёмы, которые были использованы в моём проекте: 
Гибридное хранение данных (отдельные словари для задач, целей и заметок), динамическое создание виджетов для отображения контента, кастомная отрисовка календаря, система уведомлений на QTimer с проверкой интервалов, валидация ввода через проверку полных предложений.
Стек технологий:
Python 3.12, фреймворк PyQt6, кроссплатформенная библиотека Qt6, встроенные библиотеки Python: json, datetime, sys
